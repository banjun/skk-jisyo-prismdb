# skk-jisyo-prismdb

![](docs/skk-jisyo-manakalaala.png)

プリティーシリーズのSKK辞書です。SKK方式の入力メソッドの辞書として使用できます。
元データに [PrismDB](https://github.com/prickathon/prismdb) を使っています。
